# Sample Data Directory

Place sample invoice PDF files in this directory for demos.

## Required Files

- At least one invoice PDF file (`.pdf`)

## File Naming

Files can have any name, but should be valid PDF invoices.

## Example

```
sample_data/
  ├── invoice_001.pdf
  ├── invoice_002.pdf
  └── invoice_003.pdf
```

## Note

The demos will use the first PDF file found in this directory if no specific file is specified.
